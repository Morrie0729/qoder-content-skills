# Creative Namer - Product & Brand Naming Specialist

Diagnose naming problems, generate creative name candidates, and evaluate them across sound, meaning, culture, and function. Supports both English naming conventions and Chinese creative wordplay.

## Trigger Conditions

Activate this skill when the user:
- Needs a name for a product, brand, app, project, or feature
- Has a name that "doesn't feel right" and wants diagnosis
- Says keywords like: "取名", "命名", "name this", "product name", "brand name", "起个名字", "naming", "改名"

## Part 1: Diagnosis (Why Names Fail)

When the user has an existing name that needs fixing, diagnose using these 5 states:

### N1: "Doesn't Feel Right"
Stakeholders reject names but can't articulate why.
- **Check:** Do the sounds match intended emotional tone?
- **Check:** Hidden meaning conflicts between layers?
- **Fix:** Analyze sound layer independently, test with naive audience

### N2: Names Don't Belong Together
Product family or feature set feels disjointed.
- **Check:** Consistent sound palette? Matching syllable structures?
- **Fix:** Define phoneme inventory, establish syllable templates, regenerate outliers

### N3: Forgettable
Nobody can recall the name. Blends into category noise.
- **Check:** Distinctive sound pattern? Meaning anchor? Differentiation from competitors?
- **Fix:** Add sound distinctiveness, create meaning hook, test against alternatives

### N4: Sends Wrong Signals
Audience interprets the name differently than intended.
- **Check:** Category convention alignment? Cultural references understood? Cross-language issues?
- **Fix:** Audit category sound conventions, test with target audience

### N5: Doesn't Work Practically
Misspelled, mispronounced, domain taken, hard to type/search.
- **Check:** Spelling from dictation? Typo-prone combinations? Domain/handle available?
- **Fix:** Test dictation spelling, check typo patterns, verify availability

## Part 2: The Four Analysis Layers

### Layer 1 — Sound

How a name sounds communicates before meaning is processed.

| Sound Pattern | Association | Use For |
|--------------|-------------|---------|
| Depth sounds (a, o, u, m, n) | Weight, seriousness | Authority brands |
| Light sounds (i, e, l, s) | Speed, precision, elegance | Tech, luxury |
| Power sounds (k, t, p, x) | Strength, impact | Performance, action |
| Flow sounds (l, r, w) | Movement, continuity | Creative, lifestyle |
| Tech sounds (x, z, -ix, -ex) | Modern, digital | Tech products |

| Syllable Pattern | Feel | Examples |
|-----------------|------|---------|
| CV (open) | Flowing | Sora, Kano |
| CVC (closed) | Solid | Slack, Bond |
| CVCV (balanced) | Memorable | Toyota, Roku |

### Layer 2 — Meaning

| Type | Description | Examples |
|------|-------------|---------|
| Descriptive | Says what it is | General Motors, 支付宝 |
| Metaphorical | Implies qualities | Amazon, Apple, 蚂蚁 |
| Abstract/Coined | Meaning assigned later | Kodak, Xerox |
| Portmanteau | Blended words | Pinterest, 饿了么 |

### Layer 3 — Culture & Category

| Category | English Convention | Chinese Convention |
|----------|-------------------|-------------------|
| Tech startup | Dropped vowels, -ly, -ify | 叠字(滴滴), 谐音(知乎) |
| Luxury | French/Italian sounds | 优雅汉字(兰蔻/香奈儿音译) |
| Consumer app | Short, punchy, verb-like | 动词/形容词式(去哪儿, 小红书) |
| SaaS/B2B | Compound or coined | 英文直用或中英混搭 |

### Layer 4 — Function

| Test | Pass Criterion |
|------|---------------|
| Spelling | Intuitive from pronunciation |
| Pronunciation | Intuitive from spelling |
| Typing | No awkward key combos |
| Search | Returns relevant results |
| Domain | .com or acceptable variant available |
| Social handles | @name available on key platforms |
| Voice search | Voice assistants recognize it |

## Part 3: Name Generation Strategies

### English Strategies

| # | Strategy | How | Examples |
|---|----------|-----|---------|
| 1 | **Portmanteau** | Blend two words | Pinterest, Instagram |
| 2 | **Invented** | New coined word | Spotify, Xerox |
| 3 | **Compound** | Join two words | Facebook, YouTube |
| 4 | **Creative Misspelling** | Respell for uniqueness | Lyft, Flickr, Tumblr |
| 5 | **Suffix Pattern** | Keyword + branded suffix | Shopify, Grammarly, Notion |
| 6 | **Abstract Metaphor** | Unrelated word evoking qualities | Amazon, Uber, Slack |
| 7 | **Acronym** | Initials that form a word | AWS, DALL-E |
| 8 | **Verb/Action** | Action the user takes | Zoom, Snap, Stripe |

### Chinese Creative Strategies (中文创意命名法)

| # | Strategy | How | Examples |
|---|----------|-----|---------|
| 9 | **谐音梗** | 用发音相近但含义不同/反差的字 | 饿了么(谐音"饿了吗"), 去哪儿(口语化), 闲鱼(咸鱼翻身) |
| 10 | **反差命名** | 故意用反义/意外的词制造记忆点 | 死了么(反差"饿了么"), 别买了(反向电商) |
| 11 | **网络黑话/梗** | 用网络流行语或亚文化用语 | 摸鱼(摸鱼时间), 内卷(反内卷工具) |
| 12 | **叠字法** | 重复字增加亲切感和记忆度 | 滴滴, 当当, 陌陌, 拼多多 |
| 13 | **口语化** | 用日常对话式的短语 | 去哪儿, 怎么办, 要不要 |
| 14 | **X+Y 句式** | "小X书/X多多/X了么" 等模板 | 小红书, 拼多多, 饿了么 |
| 15 | **古风/文化** | 用古诗词或典故取名 | 知乎(出自论语), 百度(众里寻他千百度) |
| 16 | **拟人/拟物** | 用动物/食物/角色拟人化 | 蚂蚁金服, 菜鸟裹裹, 飞猪 |

## Part 4: Generation Process

### Step 1: Context Gathering

Ask the user:
1. **产品是什么？** What does it do, in one sentence?
2. **目标用户？** Who uses it? (developer / consumer / business)
3. **调性偏好？** Desired vibe: fun/serious/tech/warm/edgy/minimal?
4. **语言偏好？** English only / Chinese only / both?
5. **避开什么？** Any names or styles to avoid?
6. **任何灵感？** Names they like from other products (for pattern reference)

### Step 2: Multi-Strategy Generation

Generate 20-30 candidates using at least 5 different strategies:
- If English: use strategies 1-8
- If Chinese: use strategies 9-16
- If both: mix from both sets

For each candidate, note which strategy was used.

### Step 3: Brandability Scoring

Score each candidate on a 1-10 scale across:

| Dimension | Weight | Criteria |
|-----------|--------|----------|
| Memorability | 25% | Can someone recall it after hearing once? |
| Pronounceability | 20% | Easy to say in target language(s)? |
| Spellability | 15% | Can someone type it correctly? |
| Uniqueness | 20% | Differentiated from competitors? |
| Emotional Fit | 20% | Does it evoke the right feeling? |

### Step 4: Availability Check

For top 5 candidates, use `WebSearch` to check:
- Domain availability: search `"[name].com"` or `"[name].io"`
- Existing usage: search `"[name]" app` or `"[name]" brand`
- Trademark conflicts: search `"[name]" trademark`
- Social handles: search `"@[name]" twitter` or similar

## Anti-Patterns (Bad Naming Habits)

| Anti-Pattern | Problem | Fix |
|-------------|---------|-----|
| Kitchen Sink | Trying to say everything in one name | Pick ONE primary message |
| Inside Joke | Only creators understand it | Test with naive users |
| Sound-Alike | Too similar to existing brands | Check competitors |
| Unpronounceable | Looks cool, nobody can say it | Test dictation |
| Apostrophe Catastrophe | Random punctuation for "exotic" feel | Use sparingly or not at all |
| 强行谐音 | 谐音生硬，联想不到位 | 谐音必须自然，听到就能笑/记住 |
| 蹭热点式命名 | 用过气的梗，很快过时 | 选有持久生命力的文化元素 |

## Key Insight

The best names seem obvious in retrospect — "of course it's called that." This inevitability comes from alignment across all four layers: the sound feels right for what it means, the meaning fits the context, the culture recognizes it, and it works in practice.

Bad names have hidden conflicts. Good diagnosis reveals them. Great naming resolves them into alignment.
